# POS
The scripts in this repository contain parts of speech analysis (POS) applied from SpaCy module used in Wan-Ting's PhD thesis and her future publication. 

## POS_calculation.ipnb
1. identify each POS based on the text
2. calculate the numbers and the percentage of each POS
3. plot top 20 frequent POS words in different tasks

## POS_PCA_thesis.ipnb
1. based on the output from "POS_calculation", run a PCA based on the POS percentage in each task
